# ElectronMMPlayer

## Getting Started

### Development Run
To start the app in development mode press F5 in Code or run:

```
npm run start
```

### electron-forge:

```
npm run package
npm run make
```

### builder

Creates AppImage
```
sudo npm run dist
```

## Directory Structure
src/ — source code
dist/ or out/ — packaged builds (created after running the build command)
